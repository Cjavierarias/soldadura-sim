# Soldadura Sim AR

Simulador de antorcha de soldadura usando celular y marcador ARUCO.

## Uso
1. Imprime el marcador: [assets/aruco_4x4_1000.png](assets/aruco_4x4_1000.png)
2. Abre la web en Chrome Android: [https://tunombre.github.io/soldadura-sim](https://tunombre.github.io/soldadura-sim)
3. Apunta al marcador y mueve el celular como una antorcha.

## Tecnologías
- OpenCV.js
- getUserMedia
- DeviceMotion (próximamente)
